# Shell Scripting basic 
## Commands
### ls ,pwd ,cd ,clear ,mkdir ,touch ,chmod ,ls -ltr(File Details)
### vi filename(Shell file Editor)
### cat filename(Print file)
### rm -r filename(Removes file)
### free -mg (Memory)
### nproc (CPU) 
### df -h (Disk used)
### top (For above memory,Disk and number of cpus)
